Comment (2016-11-10)

On further review of the code and implementation, the prior patch should be sufficient.

The above patch lets code work right up to the point where we call out to the services. While I've created mock versions of the FCM/GCM service, those can only replicate known states and not the level of behavior from using a live service, and cannot replicate unknown states.
